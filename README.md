Project OverviewThis repository contains the core implementation of my graduation project at Thebes Academy, which was awarded an A+ grade. The system integrates Computer Vision with Inverse Kinematics to allow a 4-DOF robotic arm to detect, track, and interact with objects in real-time.  
Core Engineering Logic Unlike simple pre-programmed arms, this project relies on dynamic mathematical calculations to determine movement:
1. Inverse Kinematics (IK) ModuleLocated in arm_kinematics.py, this module handles the translation of 3D workspace coordinates $(X, Y, Z)$ into specific joint angles. It utilizes the Law of Cosines to solve for elbow and shoulder angles, ensuring the arm reaches the target with millimeter precision.
 2. Computer Vision (CV) ProcessorThe vision_processor.py module uses OpenCV to process the video feed. It applies HSV color masking and contour detection to find the center of mass of target objects, providing the $(X, Y)$ coordinates used by the IK engine.  Technical StackLanguage: Python (Main Logic & Math).  Libraries: OpenCV (Image Processing), NumPy (Matrix Operations), PySerial (Hardware Communication).  Hardware Framework: Arduino-based control for high-torque servo motors.
 3. Key Challenges SolvedSingularity Handling: Implementing checks to prevent the arm from attempting impossible movements (out of reach).  Noise Reduction: Applying filters to the vision coordinates to prevent "jittery" arm movements caused by lighting changes.
